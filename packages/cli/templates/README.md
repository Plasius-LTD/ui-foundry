[![Changelog](https://img.shields.io/badge/changelog-md-blue.svg)](./CHANGELOG.md)

CLI scaffolding for UI Foundry projects.

Apache-2.0. ESM + CJS builds. TypeScript types included.

---

## Installation

```bash
npm install -g @plasius/ui-foundry-cli
```

---

## Usage

```bash
ui-foundry --help
ui-foundry list-templates
```

---

## Build Outputs

`npm run build` emits `dist/bin.js`, `dist/bin.cjs`, and `dist/bin.d.ts`.
